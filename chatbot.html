Work done

In my internship last summer at Macy's and Bloomingdale, i worked alongside engineers in the machine learning team to implement a Chat bot for the companyâ€™s website using the rasa bot template. A chat bot is a computer program which conducts a conversation via auditory or textual methods. Such programs are often designed to convincingly simulate how a human would behave as a conversational partner.

I was responsible for implementing the gift card, order pick up and promotions use case. In the gift card use case the chat bot was to reply any enquiries the users had about gift cards such as; gift card balance, how to buy or use the gift card etc. In the promotions use case the bot should be able to provide users with promo codes or tell the users what deals are currently ongoing in the website. To implement these use cases, I had to create and train large datasets to generate models to feed into the Chat bot.

To test the Chat bot, I had to converse with it to determine the appropriateness of its responses and try to improve its accuracy.
