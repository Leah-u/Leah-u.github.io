Blockly

Project

Work Done

Blockly is a block-based programming environment developed by Google that is often used to introduce programming to novices. Being a great tool, it is however inaccessible to visually impaired (VI) learners who usually rely on screen readers and magnification software to access information on the web.

Currently an accessible version has been implemented by Google, but the new version does not retain the format of using blocks but rather uses text see image below. My teams approach to redesigning Blockly’s user interface focuses on preserving the current drag-and-drop user interface for creating block-based program.


Accordingly, our team has extended Blockly to support keyboard navigation allowing VI users perform drag-and-drop actions using keyboard commands. Screen reader support has also been enabled using WAI-ARIA features and in some cases blocks are completely redesigned to facilitate access by VI users. Additionally, features have been added that allow VI users customize the presentation such as contrast, background color and text size. Lastly, a comment display window was added to the side of their screen to view comments in a tree view. The screen reader reads the comments as desired. See preview below.
